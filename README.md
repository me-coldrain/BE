## 구현 기능
> 1. 회원가입/로그인
- Spring Security, JWT 인증 방식으로 로그인 구현(refresh token 사용)<br>

> 2. 상품 등록(상품추가용 백엔드만 사용)<br>

> 3. 장바구니 
- 장바구니 추가, 삭제, 수량변경<br>


> 4. 리뷰 CRUD
- 상품 이미지, 리뷰 및 평점작성<br>
